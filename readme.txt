網頁使用原理及規定：
                   
                   使用者分為兩種：發現人、所屬人
                   "發現人"PO文時需要填的物品資料不含："所屬人"以及"管理員"    (所屬人、發現人關係可互換)
                   
                   當有其他使用者將自己選取為此物品的"所屬人"時，系統會過濾其身分(描述地點正確)
                   正確後系統會自動將"所屬人"登記進入item裡面，使其成為此物品的"所屬人"
                   
                   "管理員欄位"則是在"發現人"與"所屬人"配對完成後第一個進入審核的"管理員"
                   系統會將這名管理員登記進入item內
                   
                   
使用須知：
                   會員無法得到他人的資料，只有配對完成的發現人、所屬人能得到對方的通訊方式
                   所有配對系統都會儲存三個月
                   除上述條件管理員不能直接或間接公布使用者資料
